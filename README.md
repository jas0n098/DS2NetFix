# DS2NetFix

Experimental ASI plugin for Death Stranding 2 that tries to fix very high CPU usage caused by very aggressive connectivity checks.

# Usage
1. Grab the latest release and drop all files into your game directory
2. Launch the game with `WINEDLLOVERRIDES="xinput1_4=n,b"`
