# DS2NetFix
Experimental ASI plugin for Death Stranding 2 that tries to fix very high CPU usage caused by very aggressive connectivity checks when running under WINE.

# Note
Proton Experimental's bleeding-edge branch already has a similar [fix](https://github.com/ValveSoftware/wine/commit/1729f00e17e879f98f9df1f2bca86bc5d21a65df). If you don't want to use bleeding-edge, you can keep using this mod instead.

# Usage
Grab the latest release and drop all files into your game directory
